## 2. デザイン共通仕様

### 2.1 デザインコンセプト

- **テーマ**: ポケモンカードの世界観を意識した、ポップで親しみやすいデザイン
- **カラーパレット**:
  - メインカラー: `#FF6B35`（ポケモン赤に近いオレンジ）
  - サブカラー: `#4A90E2`（空・水の青）
  - アクセント: `#FFD700`（ゴールド）
  - 背景: `#1A1A2E`（ダークネイビー）/ `#F8F9FA`（ライトモード）
  - テキスト（ダーク）: `#E8E8E8`
  - カード背景: `rgba(255,255,255,0.08)`（グラスモーフィズム）
- **フォント**: Google Fonts - "Noto Sans JP"（日本語）、"Poppins"（数字・英字）
- **スタイル**: カード型UI、ガラスモーフィズム効果、ポップなアイコン

### 2.2 レスポンシブ対応

| ブレークポイント | 対象 |
|---|---|
| ～ 767px | スマートフォン |
| 768px ～ 1024px | タブレット |
| 1025px ～ | PC |

### 2.3 共通ヘッダー（参加者向け）

**ゲスト時**
```
[ASOPokeTool]  [大会一覧] [ランキング]  [ログイン]  [アカウント登録]
```

**会員ログイン時**
```

                                          +----------------+
                                          |  マイページ    |
                                          | 過去大会履歴   |

```

### 2.4 共通ヘッダー（管理者向け）

```
[ASOPokeTool 管理]  [ダッシュボード] [大会一覧]   [ログアウト]
```

**URL**: `/tournament/{id}`
**対象ユーザー**: 参加者（ゲスト・会員）

#### レイアウトイメージ（会員ログイン済みの場合）

```
+-------------------------------------------------------------+
|  [← 戻る]                                                   |
+-------------------------------------------------------------+
|  +-------------------------------------------------------+  |
|  | 🎴 第5回 ポケカ仙台オープン              [受付中🟢]   |  |
|  | 開催日: 2026/07/10（土）                               |  |
|  | 会場: 〇〇コミュニティセンター                          |  |
|  | 定員: 64名  /  現在のエントリー: 45名                  |  |
|  +-------------------------------------------------------+  |
|                                                             |
|  ==== ログイン中: たろう さん ====                           |
|  +-------------------------------------------------------+  |
|  |                                                       |  |
|  |  [⚡ 即エントリー（たろうでエントリー）]               |  |
|  |      ↑ ワンクリックで「たろう」名義でエントリー         |  |
|  |                                                       |  |
|  |  [📝 エントリー名を指定してエントリー]                 |  |
|  |      ↑ エントリー名入力画面を経由してエントリー         |  |
|  |                                                       |  |
|  +-------------------------------------------------------+  |
|                                                             |
|  ==== ゲストの場合 ====                                      |
|  +-------------------------------------------------------+  |
|  |  📝 エントリー登録                                     |  |
|  |  登録名（ニックネーム）                                 |  |
|  |  [  例: たろう                                      ]  |  |
|  |              [🎴 エントリーする]                       |  |
|  |  [👤 アカウント登録でワンクリックエントリーを使う →]    |  |
|  +-------------------------------------------------------+  |
|                                                             |
|  ==== エントリー済みの場合 ====                               |
|  +-------------------------------------------------------+  |
|  |  ✅ エントリー済み: たろう さん                         |  |
|  |     [QRコードを表示する 📱]                            |  |
|  |     [エントリーを辞退する ✗]                           |  |
|  +-------------------------------------------------------+  |
+-------------------------------------------------------------+
```

#### 操作説明

| 操作 | 動作 |
|---|---|
| [即エントリー] クリック（会員） | アカウント表示名でそのままエントリー完了。確認なし |
| [エントリー名を指定してエントリー] クリック（会員） | U-04 エントリー名入力画面へ遷移 |
| [エントリーする] クリック（ゲスト） | 登録名を入力してエントリー登録 |
| [QRコードを表示する] クリック | P-03 QRコード表示画面へ遷移 |
| [エントリーを辞退する] クリック | 確認モーダルの後、エントリーを削除 |

#### バリデーション

| 項目 | ルール |
|---|---|
| 登録名（ゲスト） | 必須。1～20文字 |
| エントリー重複 | 同一Cookie/アカウントでは1大会1エントリーのみ |

## 6. 画面遷移図

### 6.1 参加者向け遷移

```
[P-01 大会一覧]
    |
    +-- 大会詳細クリック
    v
[P-02 大会詳細・エントリー]
    |
    +-- [即エントリー]（会員のみ）→ エントリー完了 → エントリー済み表示
    |
    +-- [エントリー名を指定]（会員のみ）
    |       v
    |   [U-04 エントリー名入力] → エントリー完了
    |
    +-- [エントリーする]（ゲスト） → 名前入力 → エントリー完了
    |
    +-- [QRコードを表示]
    |       v
    |   [P-03 QRコード表示]
    |
    +-- [対戦表を見る]
    |       v
    |   [P-04 マッチング表示] → [P-05 結果登録]
    |
    +-- [ランキングを見る] → [P-06 ランキング表示]
    |
    +-- [決勝トーナメント] → [P-07 決勝T表示]

[P-08 累計ポイントランキング]  ← ヘッダーから常にアクセス可
```

### 6.2 管理者向け遷移

```
[A-01 ログイン]
    v
[A-02 管理ダッシュボード]
    |
    +-- [新規大会作成] → [A-03 大会作成]
    |
    +-- [大会管理] → [A-04 大会管理]
            |
            +-- [チェックイン管理] → [A-05]
            +-- [マッチング管理]   → [A-06]
            +-- [棄権管理]         → [A-07]
            +-- [決勝T管理]        → [A-08]
            +-- [ポイント設定]     → [A-09]
            +-- [タイマー管理]     → [A-10]
            +-- [手動選手登録]     → [A-11]
```

### 6.3 アカウント機能の遷移

```
初回アクセス（Cookieなし）
    |
    +-- ヘッダーに [ログイン] [アカウント登録] を表示
    |       |
    |       +-- [アカウント登録] → [U-01] → 自動ログイン(Cookie120日) → トップ
    |       +-- [ログイン]      → [U-02] → 自動ログイン(Cookie120日) → トップ
    |
    +-- [ゲストとして続ける] → 従来通りゲスト利用（名前入力でエントリー）

Cookie有効（次回以降）
    → 自動ログイン → ヘッダーに「たろうさん ▼」表示

ログイン中 [P-02 大会詳細]
    +-- [⚡ 即エントリー]             → アカウント表示名でワンクリック完了
    +-- [📝 エントリー名を指定して]   → [U-04] → エントリー完了

ログイン中 [ヘッダーメニュー]
    +-- [マイページ] → [U-03]
            +-- 過去大会履歴・累計ポイント閲覧
            +-- 表示名・パスワード変更
            +-- [ログアウト] → Cookie無効化 → ゲスト状態
```
